# FeedFetcher
"FeedFetcher" is a tool for displaying the latest RSS or ATOM feed in README.md, and so on.
Use after you create README.md file.

# Install

# Requirement
Your README.md file must hold these sentences.
1. Start of feed list
```md
<!--[START POSTS LIST]-->
```

1. End of feed list
```md
<!--[END POSTS LIST]-->
```
